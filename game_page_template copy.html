Countdown

Welcome to my Countdown game!

New here? The game is pretty simple!

  1. Use the "Random Vowel" button to add a random vowel to the board
  2. Use the "Random Consonant" button to add a random consonant to the board
  3. Keep pressing either button until you have a variety of vowels and consonants on the board
  4. When all 12 spaces are populated, the timer will begin
  5. You now have 30 seconds to list as many words using the letters shown as you can!
  6. The longer the word, the more points you get!
